# Formonium — Agency Landing Page

A multi-section responsive landing page built with React and Tailwind CSS.

## Sections
- Navbar (mobile hamburger menu)
- Hero (floating avatars, decorative vectors)
- About (two-column layout with image + text)
- Services (service list with dividers)
- Testimonials (quote card with floating avatars)
- Newsletter (subscription CTA)
- Footer (2×2 on mobile, single row on desktop)

## Tech Stack
- React
- Tailwind CSS
- React Icons

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px – 1024px  
- Desktop: 1024px+
